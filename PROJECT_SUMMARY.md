# Somnus CBT-I 系统扩展总结

## 项目概述
基于原有Somnus CBT-I应用，我们构建了一个完整的数据驱动睡眠认知行为疗法系统。该系统通过多维数据分析和AI智能推荐，为失眠患者提供个性化治疗任务，并根据反馈动态调整治疗方案。

## 核心扩展亮点

### 1. 数据输入模块增强
- **PSQI测评系统**：实现了完整的匹兹堡睡眠质量指数测评，包含7个维度的专业评估
- **生理数据集成**：创建了生理数据模拟服务，支持心率变异性、睡眠阶段、血氧等指标
- **治疗阶段管理**：增加了治疗阶段跟踪（评估期、强化期、巩固期、维持期）

### 2. 高级分析引擎
- **睡眠指标分析**：实时计算睡眠效率、入睡潜伏期、觉醒时间等关键指标
- **认知模式识别**：分析DBAS四个维度的认知失调模式
- **治疗效果评估**：多维度评估治疗进展和预测结果

### 3. 智能任务生成系统
- **规则引擎**：基于临床指南的规则库生成个性化任务
- **AI增强**：集成Gemini API生成更人性化的任务描述
- **优先级排序**：根据问题严重性和治疗阶段智能排序任务

### 4. 动态调整算法
- **依从性分析**：实时监测任务完成情况和反馈质量
- **策略调整**：根据治疗效果动态调整任务难度和类型
- **个性化路径**：为每位患者生成独特的治疗路径

### 5. 数据可视化增强
- **多指标趋势图**：睡眠效率、认知分数、任务完成率的综合展示
- **雷达图分析**：DBAS四个维度的认知信念可视化
- **治疗里程碑**：可视化治疗目标和达成进度

### 6. 系统架构扩展
- **模块化设计**：清晰的数据层、分析层、展示层分离
- **类型安全**：完善的TypeScript类型定义
- **服务抽象**：可替换的分析引擎和任务生成器

## 临床价值

### 循证医学基础
所有干预措施基于CBT-I临床指南：
1. **认知重建**：针对DBAS识别的认知失调
2. **睡眠限制**：基于睡眠效率数据
3. **刺激控制**：针对入睡困难
4. **放松训练**：降低生理唤醒
5. **睡眠卫生**：改善睡眠环境习惯

### 个性化治疗
- **数据驱动**：基于多维度数据的精准干预
- **动态调整**：实时响应患者反馈
- **渐进式**：随治疗进展调整难度

### 患者参与
- **直观界面**：专业的医疗级用户体验
- **即时反馈**：任务完成后的效果评估
- **进度可视化**：清晰的治疗进展展示

## 部署指南

### 本地开发
```bash
npm install
npm run dev
```

### 生产构建
```bash
npm run build
```

### GitHub Pages部署
1. 在GitHub仓库设置中启用Pages
2. 配置GEMINI_API_KEY作为仓库Secret
3. 推送代码到main分支自动部署

### 自定义部署
- 修改`vite.config.ts`中的base路径
- 调整`.github/workflows/deploy.yml`中的配置
- 配置自定义域名（如果需要）

## 总结

Somnus CBT-I系统已经从基础的数据记录应用，扩展为完整的、数据驱动的睡眠治疗平台。通过结合循证医学、智能算法和人性化设计，该系统能够为失眠患者提供个性化、动态调整的治疗方案，同时为临床医生提供强大的治疗管理工具。

系统采用模块化设计，便于进一步扩展和定制，具有良好的临床实用性和技术前瞻性。
